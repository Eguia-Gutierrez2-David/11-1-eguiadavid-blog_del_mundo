png

PNG es un formato gráfico basado en un algoritmo de compresión sin pérdida para bitmaps no sujeto a patentes.

jpg

El JPG o JPEG es un formato de imagen muy usado en el entorno digital, usualmente usado en las cámaras digitales.

gif

Graphics Interchange Format también llamado Compuserve GIF y más conocido como GIF, es un formato gráfico digital utilizado ampliamente en la World Wide Web, tanto para imágenes como para animaciones

webp

Formato de imagen moderno que proporciona compresión sin pérdida y con pérdida de datos para las imágenes más pequeñas, más ricas en la web.

svg

SVG o Gráficos Vectoriales Escalables es un estándar web para definir gráficos basados en vectores en páginas web, el estándar SVG proporciona marcado para describir rutas, formas y texto dentro de una ventana gráfica.
