CSS - Cascade Style Sheets

¿Qué es CSS?

El CSS es un lenguaje informático que especifica cómo se presentan los documentos a los usuarios. En pocas palabras es el diseño del documento.

¿Qué es una referencia por palabra clave?

Cuando se utiliza un archivo html, se puede configurar el css teniendo como referencia las palabras clave del código html. Algunas palabras clave podrían ser div, h2, p, y etc.

Ejemplo: h2{color: red;}

¿Qué es una referencia por id?

Id se refiere a la palabra en inglés identify. Los selectores de ID de CSS buscan un elemento basado en el contenido del atributo id. El atributo ID del elemento seleccionado debe coincidir exactamente con el valor dado en el selector.
