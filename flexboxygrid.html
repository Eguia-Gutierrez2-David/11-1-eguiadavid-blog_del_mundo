Flexbox

El Módulo de Caja Flexible, comúnmente llamado flexbox, fue diseñado como un modelo unidimensional de layout, y como un método que pueda ayudar a distribuir y alinear los contenidos dentro de tu página.

Flexbox también se compone de dos partes importantes, el contenedor Flex, en donde pondrás los items de flex como se ve a continuación.

En el código HTML se debe de poner ids o classes para que en el Código CSS se pueda programar el diseño. Se puede cambiar la dirección, el wrap, el contorno, el borde entre otras cosas más.

Grid

Grid es una función que, como Flexbox, permite dividir una página en secciones o áreas, pero con la diferencia de que estas áreas se pueden superponer y sobre pasar entre sí.
